This is the R-Code for the paper by Isabel Donoso, Matthias Schleuning, Daniel García and Jochen Fründ, accepted in Proceedings of the Royal Society B.

Details of the paper will follow upon publication.

As the purpose of this Repository is mostly to make the code for this paper conveniently available, it may not be further developed.

If you want to use the code and develop it further, feel free to contact me (jochen.fruend@biom.uni-freiburg.de) for changing the architecture that's more convenient for future use / new projects.
